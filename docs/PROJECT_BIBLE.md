\# MISIÓN



Desarrollar la aplicación más completa para instaladores eléctricos de Chile.



ElectroChile será una herramienta profesional destinada al diseño, cálculo, documentación y gestión de proyectos eléctricos, basada en la normativa chilena vigente y pensada para facilitar el trabajo diario de los instaladores eléctricos.



\---



\# VISIÓN



Convertirse en el software de referencia para instaladores eléctricos chilenos, permitiendo realizar desde cálculos rápidos en terreno hasta proyectos completos con generación automática de documentación técnica.



\---



\# FILOSOFÍA DEL PROYECTO



Cada función desarrollada deberá responder a una necesidad real del instalador.



No se agregarán funciones únicamente por motivos estéticos.



Toda nueva característica deberá aportar valor técnico, práctico o de productividad.



\---



\# PRINCIPIOS



1\. Simplicidad para el usuario.



2\. Código limpio.



3\. Arquitectura escalable.



4\. Diseño profesional.



5\. Alto rendimiento.



6\. Funcionar sin Internet siempre que sea posible.



7\. Basado en normativa chilena vigente.



8\. Preparado para futuras versiones Desktop y Web.



\---



\# OBJETIVOS



• Reducir el tiempo de diseño de instalaciones.



• Automatizar cálculos repetitivos.



• Disminuir errores.



• Centralizar toda la información de un proyecto.



• Generar documentación profesional.
