\# ⚡ ELECTROCHILE UI GUIDE



Versión: 1.0



\---



\# OBJETIVO



Definir la identidad visual oficial de ElectroChile.



Toda nueva pantalla deberá respetar esta guía.



\---



\# FILOSOFÍA



La interfaz debe ser:



\- Limpia

\- Profesional

\- Moderna

\- Fácil de usar en terreno

\- Visible bajo luz solar

\- Accesible
